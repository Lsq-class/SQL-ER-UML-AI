# AI TO ER-UML-Flowchart

ai生成er图、系统功能图、流程图、UML类图、三线图，sql直转er图,项目转uml类图，生成的图支持导出drawio与visio文件进行二次加工

ER图快速生成 好的画图平台可以给我们带来方便，今天我来推荐一个在线ai生成ER图平台-[https://download.moshuo.top/sqltool/](https://download.moshuo.top/sqltool/),下面是使用它的功能和使用步骤与同步的截图： 

1. sql转er图

2. AI生成UML图

3. 创建系统模块图

4. AI生成系统流程图

5. 生成图形导出成drawio或visio文件

6. er图拆分单图

7. 预览画布上修改生成图形

8. sql生成三线图

   

 #### sql转er图

把准备好的项目建表sql（也可以让ai生成功能的建表语句）放入输入栏内即可生成对应的er图。ai生成的er图也输入提示词让ai修改。

<img width="2880" height="1558" alt="image-20260526171327950" src="https://github.com/user-attachments/assets/b798afff-bbd9-4f8b-9f5b-9c6b5f63b947" />



#### AI生成UML图

打开ai提示词，输入代码执行的流程功能，系统就会自动生成UML图形。也可导入代码项目，自动转化成UML图形

<img width="2880" height="1558" alt="image-20260526171633760" src="https://github.com/user-attachments/assets/414b747a-415f-4096-9211-fc302184fa2b" />


#### 创建系统模块图

按系统功能层级输入到文本框内，就能自动生成系统功能模块图。也能让ai修改现有的功能模块，帮助检查系统功能的不合理模块。也能双击画布修改系统模块文案，也可拖拽调整模块位置。

<img width="2880" height="1558" alt="image-20260526171946447" src="https://github.com/user-attachments/assets/59c8f8ae-e049-4ff0-b350-d8c4fab4274e" />


#### AI生成系统流程图

直接输入提示词用ai生成系统流程，生成系统某个功能的流程图。

<img width="2880" height="1558" alt="image-20260526172452800" src="https://github.com/user-attachments/assets/b45c6f22-a69b-40e0-b202-f5836fdfbad0" />


生成的图形可拖动修改线段连接与双击修改流程节点的文案
<img width="2880" height="1558" alt="image-20260526172807353" src="https://github.com/user-attachments/assets/4b93ee74-00cd-4918-8f06-b031d970939d" />



#### 生成的图形都能导出成drawio或visio文件

er图导出成drawio文件在[https://drawio.io/](https://app.diagrams.net/?src=about)  的官网打开（进入官网网页后要勾选掉网格和页面视图），可以在这个网站进行二次加工er图，查漏补缺。
<img width="2880" height="1558" alt="image-20260526173441221" src="https://github.com/user-attachments/assets/c85134a2-04bb-4e9b-853b-b6cbd145a8d0" />



还能导出visio文件，可在office或wps里面导入。

<img width="2868" height="1718" alt="image-20260526174244438" src="https://github.com/user-attachments/assets/bdab62e1-aeeb-4709-916c-7b530e6ac737" />


#### er图拆分单图

点击拆分单图，可把er关联关系去除。同时还能点击画布功能按钮里面的隐藏属性，只展示实体。还能隐藏显示主键的下划线和外键的波浪线。
<img width="2880" height="1558" alt="image-20260526174600683" src="https://github.com/user-attachments/assets/10ad3a62-91fc-4034-94ab-dabd5dd04edc" />



点击工具栏的历史也可以找到历史生成的图形。

#### 预览画布上修改生成图形

双击画布文本内容可以编辑文本内容，按enter键即可保存到图形内。
<img width="1982" height="1192" alt="image-20260526175317962" src="https://github.com/user-attachments/assets/90836729-f40a-4c57-81bd-c9821d557d1b" />



#### sql生成三线图

可以用sql建表语句转成文章里面的三线图。还能配置字号、展示列等，很方便。并且支持导出到word文档内。

<img width="2880" height="1558" alt="image-20260526175534720" src="https://github.com/user-attachments/assets/8e1a6e5e-b251-4c11-9e71-f056b081d6da" />
